Computer Project:

    The manual for the computer project, Chaotic Phenomena and the Pendulum, will be handed out in class along with auxiliary information from Numerical Recipes in C, by Press et al. The only electronic files you will need to get started on the project are the two posted below:

  • RKdemo.tru
    This is a TrueBasic program which illustrates how 4th-order Runge-Kutta integration works. No modification is necessary to make it run.

  • phspshl.tru
    This is a TrueBasic program which is the starting point for the computer project. You will have to modify one line of this program to make it work, and you will modify it further in later stages of the project to get the program to model different kinds of pendulum behavior.
